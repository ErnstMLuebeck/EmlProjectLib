◆ readAdcValue()

void BatteryMonitor::readAdcValue ( float *  ValAdcRaw,
float *  ValAdcFilt 
)

Takes a defined number of ADC measurements and averages them. Afterwards a lowpass filter is applied to get a very stable ADC reading. Use a high quality multimeter and measure the corresponding voltage (before voltage devider). The ADC value as well as the multimeter voltage reading can the be used to calibrate the battery monitor.

Returns
ValAdcRaw [0..1024], average of 100 ADC readings every 1ms
ValAdcFilt [0..1024], low-pass filtered value (Ts = 1s, Tc = 5s)

◆ readBattVoltage()

float BatteryMonitor::readBattVoltage ( )

Take a defined number of battery voltage readings and average them to reduce measurement noise. The value is mapped from ADC to a true voltage (2-point calibration).

Returns
VBattRaw [V], average of NrMeasAvg voltage readings (mapped using 2-point calibration)

void BatteryMonitor::update ( )

Update all battery states: VBatt, SOC, TiOpRemain, etc. This method needs to be called every TsFilterVBatt if low-pass filtering is used. If filter is not active, the method can also be called asynchronously.
