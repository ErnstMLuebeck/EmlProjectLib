Detailed Description

Standard, parallel PID controller.

Parallel PID controller with anti-windup and filtered D-part. The class also features functions for gain scheduling (update controller gains during run-time), set value of integrator, as well as an input for feed-forward control.

Calculate PID control algorithm. Note that the actuator is typically limited to physical limits (e.g. 0..100%). If not, u_true can simply be set to u_kn1, the last controller output. If feed-foorward is not used, set it to 0.

Parameters
