Detailed Description

State Space Model (MIMO compatible) with state observer:

\( x[k] = A \cdot x[k-1] + B \cdot u[k] \)

\( y[k] = C \cdot x[k-1] \)

State observer:

\( \hat{x}[k] = (A-L\cdot C) \cdot \hat{x}[k-1] + B \cdot u[k] + L \cdot y_{sens}\)

Calculate the new observer based on the new inputs and measured plant output. The observer outputs and states can then be obtained by the respective get* functions. Standard matrix equations of a state observer:

\( \hat{x}[k] = (A-L\cdot C) \cdot \hat{x}[k-1] + B \cdot u[k] + L \cdot y_{sens}\)

u_k[-], system input vector (can also be a vector of size [1x1])
y_sens[-], measured plant output
