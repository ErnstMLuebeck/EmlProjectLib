Detailed Description

State Space Model (MIMO compatible) with state observer

State space model: x_k = A*x_kn1 + B*u_k; y_k = C*x_kn1;

State observer: x_hat_k = (A-L*C) * x_hat_kn1 + B*u_k + L*y_sens;
