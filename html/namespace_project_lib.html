◆ MatrixInvert()

int ProjectLib::MatrixInvert ( float *  A,
int  n 
)

Matrix Inversion Routine This function inverts a matrix based on the Gauss Jordan method. Specifically, it uses partial pivoting to improve numeric stability. The algorithm is drawn from those presented in NUMERICAL RECIPES: The Art of Scientific Computing. The function returns 1 on success, 0 on failure. NOTE: The argument is ALSO the result matrix, meaning the input matrix is REPLACED

Parameters
Amatrix input AND output of inversion
nnumber of rows = columns
