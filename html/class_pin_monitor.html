Detailed Description

The PinMonitor is watching a physical pin and can return a number of events: rising edge, falling edge, toggle, state,.. The state can also be debounced.

◆ update()

void PinMonitor::update ( )

Update pin monitor states. This function needs to be called regularly. The pin is sampled and a debounce counter is incremented/decremented according to the pin state. If the debounce threshold is reached, the state is set. All kind of edges are detected. There is an inhibit flag to skip the next edge. This is used after a long press, so the button release does not trigger an additional action.
